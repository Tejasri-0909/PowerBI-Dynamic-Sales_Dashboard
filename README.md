# PowerBI-Dynamic-Sales_Dashboard
This project presents an interactive Power BI dynamic dashboard built using a real-world sales dataset to analyze business performance and trends. The dashboard provides actionable insights through visually rich KPIs, charts, and slicers that enable users to explore data dynamically.

The solution focuses on sales performance analysis, regional comparison, and time-based trends, helping stakeholders make data-driven decisions efficiently.
